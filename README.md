# README - Overview/Project Explaination

In this project,i wanted to graph values obtained within a spring-damper system of a car using python(numpy,scipy,matplotlib)

In order to navigate this project,I have chosen to create this with multiple models increasing in complexity:

	-Single mass model
	-Quarter car model
	-Half car model(In Progress)

Each model has been assigned folder which includes:

	-a single .py file containing all the python code.
	-a README.md file containing all the mechanical derivatiions,self made diagrams and graphs of the results

To avoid unnecessary repetition, fundamental theory, derivations, and background explanations are presented only once 
in the introductory sections. The more advanced folders will be focused on extending and applying these concepts.
